# Wedding RSVP System 💍

A full-stack web-based Wedding RSVP system that allows guests to register, get approved or declined by an admin, and receive a QR code invitation via email.

---

## 🚀 Features

- Guest registration system
- Admin dashboard for approving/declining guests
- QR code generation for approved guests
- Email notifications (Approval & Decline)
- QR-based check-in system
- Firebase Firestore database integration

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- Firebase Firestore
- Nodemailer (Email Service)
- QRCode (QR generation)
- HTML, CSS, JavaScript (Frontend)
